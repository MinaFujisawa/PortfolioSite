ABOUT

JavaT is the app has the same concept with JavaQ, which is "for Java leaners".
This is a very simple SNS service that users be able to ask questions about Java and communicate with other people.

This is a team project with my one of classmates. See my deliverables for this project.

CHALLENGES

Since this is our very first time to make SNS service and use Firebase. It really took time to get used to it so “everything" was challenge, indeed!

Sign-In / Sign-Out

We changed firebase rule to users can see the whole feed and the detail page without sign-in (read only).
Option menu will be changed based on the user's sign-in status.
Also, it shows Sign-in dialog when you try to post and add comment if you didn’t sign-in yet.

Feed / Detail

Only owner of a Post, or a Comment, is able to see the “delete” button.

Profile

If you already have an account, it shows user’s name and user picture from Firebase. If not, empty name and default picture will be set. We use Firebase Storage to manage their pictures.
Adding TextChangedListener to the name textField, it restricts empty name and more than 20 characters name.
